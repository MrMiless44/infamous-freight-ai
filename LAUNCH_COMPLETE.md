## **ğŸ“ KNOWLEDGE BASE**

### **How to Handle Common Issues**

**API not responding?**
- Check Render logs for errors
- Verify health endpoint: /api/health
- Check database connection
- Restart service in Render Dashboard

**Payment not working?**
- Verify Stripe keys in Render env vars
- Check Stripe Dashboard for errors
- Test webhook delivery
- Check API logs for webhook events

**Web dashboard slow?**
- Check response time in Render metrics
- Optimize database queries
- Enable caching
- Consider upgrading service tier

**Database queries slow?**
- Add indexes to frequently queried columns
- Use pagination (LIMIT/OFFSET)
- Cache repeated queries
- Consider upgrading database

**Stripe webhook not delivering?**
- Verify webhook endpoint URL is correct
- Check webhook secret in Render env vars
- Test with "Send test event" in Stripe
- Check Render logs for errors
